# Liquidación Los Cardales

Sitio estático (catálogo) para la liquidación de enseres por mudanza. Se difunde por WhatsApp y grupos locales.

- **Fuente de verdad:** `inventario/inventario.json`
- **Sitio publicado:** `/docs` (GitHub Pages)
- **Operativa y reglas:** ver `CLAUDE.md`

Prioridad: velocidad de liquidación sobre maximización de precio.
